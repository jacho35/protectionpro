ProtectionPro — Help & Documentation

Welcome to ProtectionPro

ProtectionPro is a browser-based power systems engineering tool for building single-line diagrams (SLDs) and performing electrical analysis — a lightweight, web-based alternative to ETAP.

Building a Single-Line Diagram

  1. Add components — Drag components from the left sidebar palette onto the canvas. Use the search box to filter components. Categories include sources, distribution, protection, instruments, loads, DC systems, and other.
  2. Connect components — Click a connection port (small circle) on one component and drag to a port on another to draw a wire. Alternatively, press W for Wire mode.
  3. Configure properties — Click a component to select it. Edit its electrical properties in the right-hand properties panel (voltage, impedance, ratings, etc.). Sequence impedances (positive, negative, zero) can be entered for sources, cables, and motors.
  4. Run analysis — Use the Analysis, Studies, and Safety menus in the toolbar to run fault analysis, load flow, arc flash, cable sizing, and more.

Templates

Use File → Templates to start from a pre-built network: substation, industrial plant, residential distribution, or generator installation. Templates provide a fully wired and configured starting point.

Canvas Navigation

  • Pan — Middle-click drag or hold Space + left-click drag
  • Zoom — Mouse scroll wheel, or use Fit / 1:1 buttons in the toolbar
  • Select — Click a component, or drag a selection box around multiple components
  • Multi-select — Hold Shift and click to add/remove from selection
  • Dark mode — Toggle via the Dark button in the top-right toolbar

Multi-Page Diagrams

Use the page tab bar below the canvas to add, rename, and switch between diagram sheets. Double-click a tab to rename it. Drag devices between tabs to reorganize. Use off-page connectors to link electrical connections across pages — connectors with the same label are electrically joined.

Bus Resizing

Hover over a bus to reveal resize handles on each end. Drag the handles to make the bus wider or narrower. Connection ports are automatically added as the bus grows — one top/bottom pair per 40px of width. Edge ports (left/right) are available for bus tie breaker connections.

Component Grouping

Select multiple components and press Ctrl+G to group them. Groups can be selected together and moved as a unit. Press Ctrl+Shift+G to ungroup.

Wire Routing

Choose between orthogonal (right-angle), diagonal, or spline wire routing via View → Wire Routing. Double-click a wire to add bend points for custom routing. Directional flow arrows on wires show power flow direction after running load flow analysis.

Saving & Exporting

Use Ctrl+S to save your project. Projects can be saved to the server or exported in multiple formats via the Reports menu:

  • JSON — Full project data (re-importable)
  • SVG / PNG — Diagram image export
  • CSV — Analysis results as spreadsheet
  • PDF — Full report with title block, legend, and border
  • Settings Schedule — Protection device settings CSV
  • Arc Flash Labels — Printable warning labels (PDF)
  • Detailed Calculations — Step-by-step formulas and intermediate values (PDF)

Scenarios

Use Studies → Scenarios to save named network configuration snapshots. Compare scenarios side-by-side via Reports → Compare Scenarios to evaluate different operating conditions or design alternatives.

Mobile Support

ProtectionPro includes a responsive mobile interface with bottom navigation, slide-up component/analysis/properties panels, pinch-to-zoom and pan touch gestures, floating action buttons for mode switching, and tap-to-place components.

Fault Analysis (IEC 60909)

Calculates short-circuit currents at each bus per IEC 60909. Select a fault type from the dropdown or run all types at once. Results include:

  • I"k3 — Initial symmetrical 3-phase fault current (kA)
  • ip — Peak short-circuit current (kA)
  • Ib — Symmetrical breaking current (kA)
  • Ith — Thermal equivalent short-circuit current (kA)
  • I"k1 — Single line-to-ground (SLG) fault current
  • I"kLL — Line-to-line fault current
  • I"kLLG — Double line-to-ground fault current

Select a single bus before running to analyze only that bus, or run on all buses. Motor fault contribution, branch contributions, and fault current angles are included. Use View → Fault Angles & Voltage to toggle angle/voltage display on the diagram.

Voltage Depression During Fault

When a single bus is faulted, ProtectionPro calculates the retained voltage at all other buses using the Zbus impedance matrix method (Vj = 1 − Zjk/Zkk). Three time periods are computed per IEC 60909:

  • Sub-transient (0–5 cycles) — generators at Xd'', motors contributing
  • Transient (~5 cycles to 0.5s) — generators at Xd', motors decayed
  • Steady-state (>0.5s) — generators at Xd, motors fully decayed

Buses are color-coded on the diagram: green (>80%), yellow (50–80%), orange (30–50%), red (<30%). A detailed table shows all bus voltages and a motor reacceleration voltage recovery chart shows how voltage recovers after fault clearing as motors re-speed.

Load Flow (Newton-Raphson / Gauss-Seidel)

Calculates steady-state power flow through the network. Choose between Newton-Raphson (faster convergence) or Gauss-Seidel (simpler iteration) via the method dropdown. Results include bus voltages, angles, branch MW/MVAR flows, currents, and loading percentages. Voltage violations and mismatched ratings are flagged as warnings. Directional flow arrows appear on wires after analysis.

Unbalanced Load Flow

Three-phase asymmetric network analysis using the symmetrical component method. Calculates per-phase voltages and currents for unbalanced distribution systems where load or source imbalance exists. Results show phase-A/B/C voltages, unbalance factor, and negative/zero sequence components at each bus.

Time-Current Curves (TCC)

Plots protective device operating curves on a log-log time-current chart. Features include:

  • Fault overlay markers showing where fault currents intersect device curves
  • CB instantaneous line dragging — adjust magnetic pickup directly on the chart
  • Editable settings panel for selected curves (pickup, TDS, curve type)
  • Comparison mode — side-by-side before/after coordination view
  • Distance relay (21) mho characteristic curves with R-X inset diagram
  • Directional overcurrent relay (67) curves
  • User-defined curve data import via CSV
  • Drag devices between TCC tabs for custom grouping
  • CT saturation modeling showing effect of CT burden on relay operating times
  • Auto-coordination engine: topology-aware relay/CB grading with miscoordination detection

Compliance Report

Checks the network against standard requirements and generates a pass/fail/warning report. Verifications include:

  • Cable sizing adequacy (thermal, voltage drop, fault withstand)
  • Protection coordination and grading margins
  • Voltage drop limits
  • Breaking capacity adequacy
  • SANS 10142 wiring rules (LV voltage tolerance, cable protection, minimum conductor sizes, neutral earthing, maximum demand, earth fault disconnection)

Engineering Studies

ProtectionPro includes specialized engineering studies available via the Studies menu. All studies can be run individually or as a batch using Run All Studies.

Cable Sizing (IEC 60364)

Evaluates cables against three criteria per IEC 60364:

  • Thermal rating — Checks cable ampacity against load current with derating factors
  • Voltage drop — Verifies voltage drop is within acceptable limits (typically 3-5%)
  • Fault withstand — Checks adiabatic heating during short-circuit (I²t ≤ k²S²)

Results show pass/fail per criterion with recommended cable sizes. Both IEC 60364 and NEC Article 310.16 ampacity tables are supported.

Motor Starting

Analyzes voltage dip during motor starting (locked-rotor current conditions). Calculates:

  • Starting current (locked-rotor multiplier × rated current)
  • Voltage dip at the motor bus and upstream buses
  • Pass/fail against typical 15% voltage dip limit

Equipment Duty Check

Validates that protective equipment fault current ratings exceed calculated fault levels. Checks circuit breakers (breaking capacity), fuses (breaking capacity), and busbar (short-time withstand) against fault analysis results. Equipment is flagged as pass, marginal, or fail.

Load Diversity

Calculates demand factors and diversified maximum demand for the network. Uses standard demand factors by load category to determine the actual loading versus connected load. Results show demand factor, coincidence factor, and diversified demand at each bus level.

Grounding System Analysis (IEEE 80)

Designs and validates grounding grids per IEEE 80. Calculates:

  • Grid resistance — Earth resistance of the grounding grid
  • Touch voltage — Voltage between grid and 1m away, compared to tolerable limits
  • Step voltage — Voltage between feet 1m apart, compared to tolerable limits
  • Ground potential rise (GPR) — Maximum grid voltage during fault

Configure soil resistivity, grid dimensions, conductor size, and fault clearing time in the bus properties.

Run All Studies (Study Manager)

Batch-runs all enabled studies in a single operation. Select which studies to include and review combined results. Useful for comprehensive design validation.

Scenarios

Save named network configuration snapshots to compare different operating conditions — e.g., normal vs emergency, summer vs winter loading, future expansion. Use Reports → Compare Scenarios for side-by-side comparison reports.

Arc Flash Analysis (IEEE 1584-2018)

Calculates arc flash hazard at each bus per IEEE 1584-2018 and classifies PPE requirements per NFPA 70E.

What is Calculated

  • Arcing current — Current that sustains the arc (typically less than bolted fault current)
  • Incident energy — Thermal energy at the working distance (cal/cm²), determines PPE requirement
  • Arc flash boundary (AFB) — Distance where incident energy drops to 1.2 cal/cm² (onset of second-degree burn)
  • PPE category — NFPA 70E hazard category (0-4, or DANGER if >40 cal/cm²)

PPE Categories (NFPA 70E)

CategoryEnergy (cal/cm²)Required PPE
0< 1.2No arc-rated PPE required
11.2 – 4.0Arc-rated shirt, pants, safety glasses
24.0 – 8.0Arc-rated shirt, pants, flash suit hood, hard hat
38.0 – 25.0Arc flash suit, hard hat, balaclava
425.0 – 40.0Multi-layer arc flash suit
DANGER> 40.0Do not work energized

Key Bus Properties for Arc Flash

  • Working distance — Distance from the arc source to the worker (mm). Default 455mm for low-voltage switchgear (NFPA 70E Table 130.7(C)(15)(b)).
  • Electrode configuration — VCB (vertical, in box), VCBB, HCB, VOA (vertical, open air), HOA. Enclosed configurations produce higher incident energy.
  • Enclosure size — Width/depth of the enclosure (mm). Larger enclosures slightly reduce energy concentration.

How to Reduce Arc Flash Hazard

The analysis generates bus-specific recommendations. General strategies in order of effectiveness:

  1. Reduce clearing time — The single most effective measure. Incident energy is directly proportional to arc duration.
    • Enable instantaneous trip on upstream circuit breakers
    • Install arc flash detection relays (light/pressure sensors) for <35ms clearing
    • Use bus differential protection (87B) for sub-cycle clearing
    • Implement zone-selective interlocking (ZSI)
    • Use maintenance mode switches to lower trip thresholds during work
  2. Increase working distance — Incident energy follows an inverse-square relationship with distance. Moving from 455mm to 610mm significantly reduces exposure.
  3. Reduce available fault current — Use current-limiting fuses or reactors to reduce the energy source.
  4. Change electrode configuration — Open-air configurations (VOA, HOA) produce less concentrated energy than enclosed (VCB, HCB).
  5. Remote operation — Remote racking and switching eliminates personnel exposure entirely.
  6. De-energize — For Category 4 or DANGER, de-energize before work per NFPA 70E §130.2.

IEEE 1584-2018 Applicability

Valid for: 208V to 15kV, 3-phase, 50/60 Hz, bolted fault current 500A to 106kA, gaps 6.35 to 76.2mm. For voltages >15kV, the Ralph Lee theoretical model is used. Below 240V with fault current <500A, arc flash hazard is generally minimal.

DC Arc Flash Analysis

Calculates DC arc flash hazard using the Stokes & Oppenlander method with DGUV-I 203-077 guidelines. Available via Safety → DC Arc Flash. Applicable to DC systems including battery rooms, UPS systems, rectifiers, and solar PV DC buses.

  • DC arcing current — Sustained DC arc current based on system voltage and gap
  • Incident energy — Thermal energy at working distance (cal/cm²)
  • PPE category — Hazard category per NFPA 70E
  • Arc flash boundary — Safe working distance
